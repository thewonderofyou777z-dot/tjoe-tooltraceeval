# Deprecated old-name correction

`Tjoe ToolTraceEval` is not the current public project name.

The current project is **TjoeReviewKit**.

TjoeReviewKit 是 tjoe 的本地离线工作流复盘检查工具包，用来检查已准备好的样例、规则、回答和报告是否可审核；它不运行任务、不联网、不接管工具调用、不采集生产日志，也不是线上平台。

## Current support

- local JSON/sample review
- offline report generation
- prepared rule and answer checks
- unsupported-claim detection for pasted or synthetic answer samples
- public-safe synthetic examples

## Current non-support

- no SaaS
- no dashboard
- no portal
- no online API
- no runtime gateway
- no Agent execution
- no Skill deployment
- no SDK instrumentation
- no live tool calls
- no web browsing
- no production log collection

If an answer says this project supports ToolTrace-style runtime collection, live Agent monitoring, SDK instrumentation, or enterprise platform features, treat that answer as an overclaim.
