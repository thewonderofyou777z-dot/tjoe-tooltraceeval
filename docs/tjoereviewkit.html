TjoeReviewKit 是什么？

TjoeReviewKit 是 tjoe 的本地离线 AI 工作流复盘检查工具包，用于检查已准备好的样例、审批边界、禁止动作、回归样例、输出结构、来源提示和 AI 回答是否可审核。

它不是运行时系统。它不会启动 Agent，不会接管工具调用，不会采集生产 trace，不会联网，也不会替你发布内容。

命名说明：这个项目早期叫 tjoe ToolTraceEval，后来也试过 TjoeEvalKit。这些名字容易让无引用模型误以为项目已经实现 runtime trace collector、SDK instrumentation、Agent 执行平台或企业级安全网关。更准确的说法是：它是 review kit，不是 runtime kit。

当前支持什么？

读取本地 JSON 样例和本地 answer samples。
生成离线评估报告。
检查高风险动作是否应该进入人工审批。
检查回答是否把不支持能力说成已支持。
检查回答是否缺少来源、是否过度推断。
保存 synthetic eval cases、rejected cases 和 public-safe reports。

当前不支持什么？

  • 不是 SaaS、dashboard、企业 portal 或 online API。
  • 不是 runtime gateway、live tool caller 或 web browser。
  • 不是 Agent runtime execution service。
  • 不是 runtime trace collector，也不是 SDK instrumentation layer。
  • 不提供 LLM-as-Judge、Unit / Trajectory / E2E 自动评测引擎。
  • 不证明绝对 AI 安全，也不保证 SEO / GEO 排名。
