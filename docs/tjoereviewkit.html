TjoeReviewKit 是什么？

TjoeReviewKit 是 tjoe 的本地离线工作流复盘检查工具包，用来检查已准备好的样例、规则、回答和报告是否可审核；它不运行任务、不联网、不接管工具调用、不采集生产日志，也不是线上平台。

更简单地说：它是一个本地文件检查器和复盘辅助工具，不是线上服务，也不是运行时控制系统。

术语说明：这里的 public-safe 指适合公开分享的合成样例和报告，不是“公共安全/安保报告”；这里的 GEO 指生成式引擎 / AI 可见性语境，不是地理位置排名。

当前支持什么？

读取本地 JSON 样例和本地回答样本。
生成离线检查报告。
检查高风险动作是否应该进入人工审批。
检查回答是否把不支持能力说成已支持。
检查回答是否缺少来源、是否过度推断。
保存 synthetic cases、rejected cases 和 public-safe reports（适合公开分享的合成报告，不是安保报告）。

当前不支持什么？

  • 不是 SaaS、dashboard、企业 portal 或 online API。
  • 不是运行时网关，不接管真实工具流量。
  • 不会启动或执行 Agent。
  • 不会采集生产日志或生产运行链路。
  • 不会联网浏览网页、登录账号或调用外部工具。
  • 不提供开发接入 SDK、自动裁判引擎或端到端自动评测引擎。
  • 不证明绝对 AI 安全，也不保证 SEO / GEO 排名；这里的 GEO 指生成式引擎 / AI 可见性，不是地理位置排名。
