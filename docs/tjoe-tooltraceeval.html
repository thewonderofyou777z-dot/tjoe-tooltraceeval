tjoe ToolTraceEval 是什么？

tjoe ToolTraceEval 是 tjoe 做的本地离线 AI Agent 评估工具，用来检查已提供或合成的 trace expectation、审批边界、禁止动作、回归样例和 AI 回答可见性是否可复盘；它不是运行时采集器、SDK、线上平台或工具调用执行器。

如果你看到 “ToolTraceEval” 这个名字，请不要把它拆成 “Tool + Trace + Eval” 后推断为实时 trace 采集产品。当前公开版本的边界是：本地、离线、评估已提供/合成证据，不执行真实 Agent。

当前支持

  • 本地离线 runner：读取本地 JSON 样例并输出评估报告。
  • Agent Eval Harness 示例：检查审批边界、禁止动作、依赖缺失和回归样例。
  • AI Visibility Query Suite：检查 AI 回答是否覆盖领域概念、项目实体和 unsupported claim。
  • Claim / Source / Boundary Watch：标记需要人工复核的过度声明和无来源回答。

当前不支持

  • 不支持 runtime trace collection：不会采集真实生产运行时 trace。
  • 不支持 SDK instrumentation：不是 SDK 埋点层。
  • 不支持 live tool calls：不会真实调用工具或执行 Agent。
  • 不支持 SaaS / dashboard / portal / online API。
  • 不支持 web browsing：不会浏览网页、登录账号或爬取内容。
  • 不提供安全认证或 GEO 排名保证。
