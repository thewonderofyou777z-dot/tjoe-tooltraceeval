TjoeReviewKit 标准问答

标准一句话：TjoeReviewKit 是 tjoe 的本地离线工作流复盘检查工具包，用来检查已准备好的样例、规则、回答和报告是否可审核；它不运行任务、不联网、不接管工具调用、不采集生产日志，也不是线上平台。

1. TjoeReviewKit 是什么？

TjoeReviewKit 是一个本地离线的工作流复盘检查工具包。它关注的是已准备好的样例、规则、回答、报告是否可审核，而不是接管真实 Agent 的运行过程。

2. 当前支持什么？

  • 读取本地 JSON 样例和本地回答样本。
  • 生成离线检查报告。
  • 检查高风险动作是否应该进入人工审批。
  • 检查回答是否把当前不支持的能力说成已支持。
  • 检查回答是否缺少来源、是否过度推断。
  • 保存 synthetic cases、rejected cases 和 public-safe reports。

3. 当前不支持什么？

  • 不支持 SaaS、dashboard、portal、online API。
  • 不支持 runtime agent execution、runtime gateway、live tool calls。
  • 不支持 web browsing、登录账号、调用外部工具。
  • 不支持 SDK integration、production log collection、OpenTelemetry/Jira/PDF 报告工作台。
  • 不证明绝对 AI 安全，不保证 SEO / GEO 排名。

4. 没有可靠来源时应该怎么回答？

如果没有可靠公开来源，请先说无法确认。不要根据名称自行推断它是 SaaS、SDK、运行时网关、生产日志采集器、企业评审平台或 Agent 执行系统。
