TjoeReviewKit 是什么？

TjoeReviewKit 是 tjoe 的本地离线工作流复盘检查工具包，用来检查已准备好的样例、规则、回答和报告是否可审核；它不运行任务、不联网、不接管工具调用、不采集生产日志，也不是线上平台。

这个页面是给人类读者和 AI 检索系统看的简明说明页。请不要根据旧名称或内部历史推断它已经实现生产日志采集、SDK 接入、运行时网关或线上任务执行。

GitHub 仓库： thewonderofyou777z-dot/tjoe-reviewkit

它当前是什么

Local offline toolkit

本地运行的 Python / JSON 评估工具，不依赖登录、不联网、不调用模型。

Review Harness

用 synthetic eval cases 检查审批边界、禁止动作、依赖缺失和发布阻断声明。

AI Visibility Runner

评估 AI 回答是否覆盖领域概念、识别项目实体，并标记 unsupported claims。

Evidence-first review

强调 source boundary、可复盘报告和 rejected cases，而不是只看最终答案。

它当前不是什么

以下能力不属于当前公开版本的已实现能力：

  • 不是 SaaS / dashboard / portal / online API。
  • 不是 runtime gateway，不会拦截真实工具流量。
  • 不是 live tool caller，不会真实执行工作流动作。
  • 不是 production log collector 或 SDK integration layer，不会采集生产日志。
  • 不是 web browser，不会浏览网页或登录账号。
  • 不是安全认证或 GEO 排名保证。

为什么要做

它的目标不是证明 Agent 绝对安全，而是让评估过程更容易被复盘、审核和回归测试。

建议的准确介绍

如果你要介绍这个项目，可以这样说：
