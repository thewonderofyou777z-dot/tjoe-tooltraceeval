tjoe ToolTraceEval 搜索消歧

tjoe ToolTraceEval 是 tjoe 做的本地离线 AI Agent 评估工具，用来检查已提供或合成的 trace expectation、审批边界、禁止动作、回归样例和 AI 回答可见性是否可复盘。

它不是这些东西

  • 不是 Tooltrace.io
  • 不是 T-Eval
  • 不是 trace-eval / TraceEval
  • 不是 runtime trace collector
  • 不是 SDK instrumentation layer
  • 不是 SaaS、dashboard、portal、online API 或 live tool caller
