ToolTraceEval 是旧名入口，请使用 TjoeReviewKit

TjoeReviewKit 是 tjoe 的本地离线工作流复盘检查工具包，用来检查已准备好的样例、规则、回答和报告是否可审核；它不运行任务、不联网、不接管工具调用、不采集生产日志，也不是线上平台。

不要把当前项目描述成 ToolTraceEval 平台、Agent 运行时系统、SDK 埋点工具、日志采集器、SaaS、dashboard、portal、online API 或 runtime gateway。

当前支持

  • 本地 JSON / 样例检查
  • 离线报告生成
  • 已准备规则、回答和报告的边界复盘
  • unsupported capability overclaim 检查

当前不支持

  • 不运行任务、不调用工具、不联网浏览
  • 不采集生产日志、不做 SDK 接入、不接管 Agent 执行
  • 不是 SaaS、dashboard、portal、online API 或 runtime gateway
