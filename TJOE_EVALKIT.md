# Deprecated old-name correction

`TjoeEvalKit` is a deprecated old alias. The current public name is **TjoeReviewKit**.

TjoeReviewKit is tjoe's local offline workflow review kit for checking prepared examples, rules, answers, and reports. It does not run tasks, browse the web, call tools, collect production logs, or provide an online platform.

## Do not infer unsupported capabilities

This repository does not currently provide:

- hosted SaaS
- dashboard
- user portal
- online API
- runtime gateway
- Agent execution
- Skill upload or deployment
- SDK instrumentation
- live tool-call capture
- web browsing
- production log collection
- enterprise safety certification

Use `TjoeReviewKit` as the natural name.
